Legal Variable Names:

  • firstName
  • age1
  • $price
  • _totalAmount
  • userCity

Illegal Variable Names:

  • 1srName
  • user name
  • @price
  • var
  • my-age

"Rules for naming JS variables"

Variable names can only contain letters, numbers, $(dollar sign) and _(underscore).

Variables must begin with a $(dollar sign), _(underscore) or letter.

Variables names are case sensitive.

Variables names should not be JS Keywords.
